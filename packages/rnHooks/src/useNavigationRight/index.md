# useNavigationRight

自定义导航右上角组件

## 代码演示

### 基础示例

```typescript
import { useNavigationRight } from '@fruits-chain/hooks-laba-rn'

useNavigationRight(<CustomCom onPress={() => {}} />)
```

### 参数

| 参数              | 说明                 | 类型              | 默认值 |
| ----------------- | -------------------- | ----------------- | ------ |
| NavRightCustomCom | 自定义导航右上角组件 | `React.ReactNode` | -      |
