# useNavigationResume

导航/路由/页面再次激活或离开后回来时触发

## API

```typescript
useNavigationResume(fn: () => void)
```

### 参数

| 参数 | 说明                       | 类型         | 默认值 |
| ---- | -------------------------- | ------------ | ------ |
| fn   | 页面再次聚焦触发的回调函数 | `() => void` | -      |
